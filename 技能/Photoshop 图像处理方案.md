# Photoshop 图像处理方案

## 处理流程
1. Lr中初步调整后载入PS
- 全选照片-编辑-自动对齐图层（如有多张图像）；或者手动对齐图层
- 合并曝光（如有多张图像）
- 评估图像
- 图像修复
- 对比度调整
- 颜色调整
- 加虚
- 亮度调整
- 降噪
- 加锐
- 导出

## 细节操作

#### LR初步调节

- 调整白平衡
- 初步降噪：颜色降噪25是可以的，亮度降噪可以做一点，但是保证细节丰富区域的细节不要丢
- 初步锐化：稍微给一点锐化

#### 合并曝光

- 多张顺序曝光图片进行曝光合并
    1. 图层排列方式：把曝光从低到高对照片进行排列
    - 从顶部照片开始，根据情况生成黑色/白色蒙版，用白色/黑色刷子刷（流量、透明度调整——各10~30%）
    - 不断合并图层，依次处理剩下的图层
- 多张复杂曝光图片进行曝光合并
    1. 选择一张大部分曝光正常的照片，生成亮部、暗部、中灰部亮度蒙版（Tony Kuyper插件）
    - 图层排列方式：把曝光最合理的照片图层放到最上层，然后按照从上到下处理的顺序，把需要刷的图层依次排列
    - 两种蒙版刷法（可以任选一种作为习惯）：
        - 把需要刷的图层放到第二层，对第一层生成白色蒙版，用黑色刷子刷（流量、透明度调整——各10~30%）
        - 把需要刷的图层放到第一层，对第一层生成黑色蒙版，用白色刷子刷（流量、透明度调整——各10~30%）
    - 不断合并图层，依次处理剩下的图层
- Tips：
    - 亮度蒙版主要用于大光比的场景，如果场景曝光本身处于中灰状态，则不用借助亮度蒙版
    - 可以用曝光合并做焦点合并

#### 图像修复

- 污点修复画笔工具 + 蒙版
- 修复画笔工具 + 蒙版
- 修补工具 + 蒙版

#### 对比度调整

1. 整体对比度调整：亮度蒙版中选择某个中灰度选区，拉曲线调整对比度（只对中灰度区域进行对比度调整，避免影响亮部区域和暗部区域） + 蒙版
2. 局部影调对比度调整：滤镜 - Nik Software - Color Efex Pro 4 - Tonal Contrast + 蒙版
3. 局部细节加强：滤镜 - Nik Software - Color Efex Pro 4 - Detail Extractor + 蒙版

#### 颜色调整

- 整体颜色调整（颜色蒙版）
    1. 生成颜色蒙版
        1. 复制当前图层
        - 调整图层 - 可选颜色 - 把每种颜色的黑色调整为-100%，白色、中性色、黑色调整为100% - 绝对 ； 通道 - 按住Comd点击RGB通道生成选区 - 保存为alpha通道（即为过饱和颜色蒙版）
        - 调整图层 - 可选颜色 - 把每种颜色的黑色调整为100%，白色、中性色、黑色调整为-100% - 绝对 ；通道 - 按住Comd点击RGB通道生成选区 - 保存为alpha通道（即为欠-饱和颜色蒙版）
        - 记录为动作（也可以加上色相/饱和度调整图层，直接生成更进一步的颜色蒙版调整图层）
        - Tips：可以通过Comd+Opt+Shift点击已生成的颜色蒙版，将会自动选择更亮的部分，从而获得更局限的颜色蒙版（此操作类似亮度蒙版）
    - 通过动作生成过饱和颜色蒙版调整图层，降低饱和度（饱和度-15左右） + 蒙版
    - 通过动作生成欠饱和颜色蒙版调整图层，提升饱和度（饱和度+25左右） + 蒙版
- 局部颜色调整
    - 色相/饱和度 + 蒙版
    - 照片滤镜（如冷却滤镜） + 蒙版
    - 滤镜 - Nik Software - Color Efex Pro 4 （如 Skylight Filter） + 蒙版
    - 渐变映射 + 蒙版
    - 色彩平衡 + 蒙版
- Tips
    - 颜色调整时，一般先调局部颜色，再调整体颜色
    - 在银河上加一点暖色可以突出一点紫色
    - 一般来说可以给有光的地方加天光色，阴影部加蓝色
    - 颜色调整，加橙色的同时可以考虑加一点红；加蓝色的同时可以考虑加一点青、绿。这样可以使颜色厚重一点，比较正

#### 加虚

- 滤镜 - 其它 - 高反差保留，选择较大半径（比如60），混合模式选择柔光，Comd + I 反转效果 + 蒙版
- Nik Software - Color Efex Pro 4 - Classical Soft Focus + 蒙版
- Tips：
    - 加虚同时会提亮，一般针对细节不充分或者远方进行处理

#### 亮度调整

- 生成新的蒙版（50%灰） - 混合模式混合模式 - 叠加 - 减淡/加深工具（曝光度10%）

#### 降噪

- 滤镜 - Nik Software - Dfine 2 + 蒙版
- Tips
    - 降噪理念：对细节部分少的地方进行降噪，细节部分多的地方不处理

#### 加锐

- 对颜色加锐很危险，所以避免对颜色进行加锐，仅对明度进行加锐：
    - 方案1：复制图层 - 混合方式 - 明度 - 加锐（滤镜 - 锐化 - 智能锐化，数量100%，半径1）；蒙版+刷子
    - 方案2：
        1. 图像 - 模式 - LAB颜色
        - 通道 - 明度通道 - 加锐（滤镜 - 锐化 - 智能锐化，数量100%，半径1）
        - 通道 - a通道 - 模糊 - 高斯模糊（2.5像素）
        - 通道 - b 通道 - 模糊 - 高斯模糊（2.5像素）
        - 蒙版 + 刷子
        - 图像 - 模式 - RGB颜色
- 对小尺寸照片进行加锐
    - 把尺寸调到需要加锐尺寸的2倍（长、宽）：图像 - 图像大小（分辨率72dpi）
    - 连续进行3-4次加锐：滤镜 - 锐化 - 智能锐化（数量100%，半径0.2，移去镜头模糊）
    - 混合方式 - 柔光
    - 把尺寸调到目标尺寸（分辨率72dpi）
- Tips：
    - 加锐理念：对细节部分加锐，细节部分少、需要柔化的地方不加锐

#### 导出

1. 另存一份16位的tif副本，用于原图保留
- 合并 - 图像通道改成8位 - 编辑 - 转换为配置文件（Prophoto -> sRGB）- 导出

#### Tips

- 效果调整手段
    - 效果本身的设置强度
    - 笔刷的强度
    - 图层本身的不透明度
    - 混合模式
    - 高斯模糊
- 为了使过渡自然，刷蒙版对一小块区域处理后，可以用大刷子刷一刷周围
- 在曝光合并的过程中，可以对各图层的亮度、颜色进行分别的调节
- 将图像转化为智能对象，可以使多个滤镜共用蒙版，且随时对滤镜效果进行调节
- 前期拍摄向右曝光：曝光补偿 +0.3-0.7，甚至1.0
- 直接将多张图片载入为图层：文件 - 脚本 - 载入堆栈
- 做旧效果：Nik Software - Color Efex Pro 4 - Bleach Bypass + 颜色调整
- 做过曝效果的注意点：
    - 通过曲线做预过曝
    - 加天光色，注意过程中不要发生偏色，可以将底色的饱和度先降掉
    - 用加虚的方式来模拟布光
    - 通过色彩平衡图层对高光部分进行加暖
    - 注意颜色过渡

- 移动 V
- 裁剪 C
- 吸管 I
- 画笔 B
- 文字 T
- 临时激活抓手 Space
- 临时激活缩放工具 Space + Comd/ Opt
- 切换前景、背景色 X
- 用背景色填充 Comd + BackSpace
- 回到历史记录上一层 Comd + Opt + Z
- 隐藏选区 Comd + H
- 反相 Comd + I
- 改变画笔大小 [/ ]
- 移动图层 Comd + [/ ]
- 图层/蒙版操作
    - 移动图层时按住Shift可以对齐
    - 查看蒙版范围 按住Opt左键点击
    - 暂时关闭蒙版 按住Shift左键点击
    - 以背景色新建蒙版 按住Opt新建蒙版
    - 将蒙版载入为选区 按住Comd左键点击
    - 将调整图层只作用在某个图层上 按住Opt拖动
    - 复制图层 Comd + J
    - 向下合并图层：Comd + E
    - 生成新的整合图层：Comd + Shift + Opt + E
    - 移动图层 Comd + [/ ]
    - 置顶/置底图层 Comd + Shift + [/ ]
